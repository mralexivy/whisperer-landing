Speaking naturally is fast, but it produces messy text. Filler words, rambling sentences, casual phrasing. Fine for talking, not great for professional writing.

Whisperer can clean this up automatically before the text gets inserted. An LLM runs locally on your Mac and transforms your raw dictation into something polished.

No cloud. No API keys. No per-word charges. Just your Mac doing the processing.

## The Built-In Modes

### Rewrite

Takes spoken text and makes it read like written text. Cuts the filler, fixes run-on sentences, tightens things up.

**Before:** "so basically what I was thinking is that we should probably go ahead and implement the new caching layer because it would make things faster"

**After:** "We should implement the new caching layer to improve performance."

### Translate

Speak in one language, get text in another. All offline. The translation quality is decent for most common language pairs.

**Before:** "Please send me the quarterly report by Friday afternoon."

**After:** "Bitte senden Sie mir den Quartalsbericht bis Freitagnachmittag."

### Format

Applies Markdown formatting with headers, bullets, and structure.

**Before:** "the project has three phases first research second development third deployment each phase takes about two weeks"

**After:**
```
# Project Phases

1. **Research** - 2 weeks
2. **Development** - 2 weeks
3. **Deployment** - 2 weeks
```

### Summarize

Condenses long dictation into key points.

### Grammar

Fixes grammar and punctuation. Doesn't change meaning, just cleans up the rough edges.

### List Format

Detects when you're listing things and formats them as actual lists.

### Coding

Turns spoken descriptions into technical docs or code comments.

### Email

Adds a greeting and sign-off, formats as a proper email.

### Creative

Makes the language more colorful. Useful for marketing copy.

### Custom

Write your own prompt. The LLM follows your instructions.

## Rewriting Existing Text

You can also rewrite text you've already written. Press Option+Shift+Tab to run your clipboard through the active AI mode.

1. Select text in any app
2. Copy it (Cmd+C)
3. Press Option+Shift+Tab
4. The AI-processed version replaces your clipboard

Handy for cleaning up notes, reformatting text, or translating things you've already written.

## History and Undo

Every transformation is saved with before/after text. Made a mistake? Undo and get the original back.

## Privacy

The LLM runs on your Mac. Your text stays on your Mac. No API keys, no cloud servers, no subscription fees.
