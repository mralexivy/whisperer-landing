You can spend $249 on a Mac dictation app, or you can spend $2.99. The expensive ones aren't necessarily better. Here's what's available under $30.

## Prices

| App | Price | Model | Offline |
|-----|-------|-------|---------|
| Apple Dictation | Free | Built-in | Yes (Apple Silicon) |
| Whisperer Base | $2.99 | One-time | Yes |
| Whisperer Pro | $14.99 | Lifetime | Yes |
| JesType | €14.95 | One-time | Yes |
| VoiceInk | ~$25 | One-time | Yes |

All under $30. Compare that to Superwhisper ($249) or Wispr Flow ($120+/year).

## 1. Apple Dictation

**Price:** Free | **Offline:** Yes (Apple Silicon) | **Best for:** Casual notes

Apple's built-in option. It's gotten better with recent macOS versions. On-device on Apple Silicon.

<FeatureGrid>
<FeatureCard title="Good" icon="star">
Free. No setup. Works everywhere. On-device. Decent for casual English.
</FeatureCard>
<FeatureCard title="Missing" icon="x">
No Code Mode. No profiles. Autocorrect mangles technical terms. No dictionary. No file transcription.
</FeatureCard>
</FeatureGrid>

**Bottom line:** Fine for quick notes. You'll hit limits fast if you do anything beyond that.

## 2. Whisperer

**Price:** $2.99 base, $14.99 Pro Pack (once) | **Offline:** Yes | **Best for:** Developers, power users, anyone who wants features without subscriptions

Menu-bar app. Hold Fn, speak, release. Text goes wherever your cursor is. Pro Pack adds Code Mode, per-app profiles, dictionary, AI processing.

<Callout type="tip">
$14.99 gets you features that cost $99-$249 elsewhere: Code Mode, per-app profiles, three engines, AI post-processing.
</Callout>

### $2.99 (Base)

- Hold-to-record and toggle modes
- 3 engines (Whisper, NVIDIA, Apple Speech)
- 100+ languages
- Live preview while speaking
- File transcription
- Filler word removal
- Transcription history
- Fully offline

### $14.99 (Pro Pack)

Everything above, plus:
- **Code Mode** — camelCase, snake_case, symbols
- **Per-app profiles** — automatic switching
- **Dictionary** — custom terms
- **AI processing** — 10 offline modes
- **Better insertion** — clipboard-safe
- **Stats** — WPM, charts

**Bottom line:** Most features for the price. Code Mode and per-app profiles are the main draws.

## 3. JesType

**Price:** 14.95 euros once | **Offline:** Yes | **Best for:** Simple dictation

Clean interface. Supports Whisper and Moonshine models.

<FeatureGrid>
<FeatureCard title="Good" icon="star">
Simple. Offline. Moonshine support. Same price range as Whisperer.
</FeatureCard>
<FeatureCard title="Missing" icon="x">
No Code Mode. No profiles. No AI processing. No file transcription. One engine.
</FeatureCard>
</FeatureGrid>

**Bottom line:** Competent but basic. Same price as Whisperer Pro, fewer features. Makes sense if you really want minimal UI.

## 4. VoiceInk

**Price:** ~$25 once | **Offline:** Yes | **Best for:** Clean design

Focused on simplicity. Uses Whisper, offline.

**Bottom line:** Solid. But $25 for fewer features than Whisperer at $14.99. The design is nice though.

## 5. Spokenly

**Price:** Free + premium tiers | **Offline:** Partial | **Best for:** iOS users

Free tier available. Has an iOS companion app. Still relatively new.

**Bottom line:** Good if you need the iOS app. Mac features aren't as deep as Whisperer yet.
