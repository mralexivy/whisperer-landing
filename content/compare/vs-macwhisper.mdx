Both apps use Whisper. Both run on your Mac. But they're for different things entirely.

## Different Tools for Different Problems

<Callout type="info">
**Whisperer** is for live dictation. You speak, text appears in whatever app you're using. Real-time.

**MacWhisper** is for transcribing files. You drag in a recording, it produces a transcript. Batch processing.

| | Whisperer | MacWhisper |
|---|----------------------|---------------------------|
| Input | Your microphone, live | Audio/video files |
| Output | Text in the active app | Transcript in MacWhisper |
| When | As you speak | After the recording exists |
| Use case | Writing emails, code, messages | Transcribing meetings, podcasts |
</Callout>

## Can You Use Both?

<Callout type="tip">
Sure. They don't conflict. Plenty of people use Whisperer for daily dictation and MacWhisper for transcribing meeting recordings. Different tools, different jobs.
</Callout>

## The Short Version

Not really a "which is better" question. They do different things.

**Live dictation?** Whisperer.
**Transcribe files?** MacWhisper.
**Both?** Both.

## Which One

<Callout type="tip">
**Get Whisperer if:**
- You need live dictation (MacWhisper can't do this at all)
- You want live dictation and file transcription in one app
- You write code and need [Code Mode](/code-mode)
- Budget matters: $2.99-14.99 vs ~$64

**Get MacWhisper if:**
- File transcription is all you need
- You need SRT/VTT subtitle export
- You batch-process lots of recordings
- You never need live dictation
</Callout>
