Whisperer and MacWhisper both use whisper.cpp on your Mac, but they solve fundamentally different problems. Understanding the difference will save you from buying the wrong tool.

## The Key Difference

<Callout type="info">
**Whisperer** is a **dictation** tool — you speak, and text is inserted live into whatever app you're using. It's for real-time voice input.

**MacWhisper** is a **transcription** tool — you feed it audio files (recordings, meetings, podcasts), and it produces a text transcript. It's for batch processing.

These are different workflows:

| | Whisperer (Dictation) | MacWhisper (Transcription) |
|---|----------------------|---------------------------|
| Input | Live microphone | Audio/video files |
| Output | Text inserted into active app | Transcript in MacWhisper's UI |
| Timing | Real-time, as you speak | After-the-fact processing |
| Use case | Writing emails, code, messages | Transcribing meetings, interviews |
</Callout>

## When You Need Whisperer

<FeatureGrid>
<FeatureCard title="Live Dictation" icon="mic">
You want to **speak and have text appear** in VS Code, Slack, Gmail, Terminal, or any app.
</FeatureCard>

<FeatureCard title="Code Mode" icon="code">
You need **Code Mode** for dictating programming syntax.
</FeatureCard>

<FeatureCard title="Real-Time Input" icon="zap">
You want **real-time dictation** with a hold-to-talk shortcut.
</FeatureCard>

<FeatureCard title="Per-App Profiles" icon="target">

</FeatureCard>
</FeatureGrid>

## When You Need MacWhisper

<FeatureGrid>

<FeatureCard title="Subtitle Generation" icon="file-text">

</FeatureCard>

<FeatureCard title="Podcast Transcription" icon="headphones">

<FeatureCard title="Batch Processing" icon="package">

## Can You Use Both?

<Callout type="tip">
Yes — and many users do. They're complementary tools:

- **Whisperer** for live dictation throughout the day (writing code, emails, messages)
- **MacWhisper** for transcribing recorded meetings and generating subtitles

Both use whisper.cpp, both run offline, and they don't conflict with each other.
</Callout>

## The Verdict

This isn't about which tool is "better" — they solve different problems.

**Need live dictation?** Get Whisperer.
**Need file transcription?** Get MacWhisper.
**Need both?** Get both — they work great together.
