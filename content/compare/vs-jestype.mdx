JesType is a simple offline dictation app for Mac. It runs Whisper and the Moonshine model locally and costs about the same as Whisperer (around $15). The apps solve the same basic problem but take different approaches.

Same ballpark price. Both one-time purchases. The difference is features: Whisperer adds Code Mode, per-app profiles, and AI post-processing. JesType keeps things minimal.

## Where Whisperer Wins

<FeatureGrid>
<FeatureCard title="Code Mode" icon="code">
JesType doesn't have developer features. Whisperer lets you dictate `camelCase`, `snake_case`, parentheses, brackets, arrows, semicolons. If you write code, this is the gap.
</FeatureCard>

<FeatureCard title="Three Engines" icon="cpu">
Whisperer has Whisper (GPU), NVIDIA (Neural Engine), and Apple Speech. JesType has Whisper and Moonshine. The NVIDIA engine is fast because it runs on the Neural Engine, leaving your GPU free.
</FeatureCard>

<FeatureCard title="AI Post-Processing" icon="sparkles">
Whisperer can rewrite, translate, format, summarize, and fix grammar after transcription. Ten modes, all offline. JesType just transcribes.
</FeatureCard>

<FeatureCard title="Per-App Profiles" icon="target">
Whisperer switches modes automatically. Code Mode in VS Code, normal text in Slack. JesType uses the same settings everywhere.
</FeatureCard>
</FeatureGrid>

## Where JesType Wins

### Moonshine Support
JesType runs the Moonshine model. Whisperer doesn't. Moonshine is smaller and faster than most Whisper variants, so if model size matters, JesType has an edge there.

### Simplicity
JesType does less. Some people want that. No modes to configure, no profiles to set up. Install and go.

### Good SEO
They rank well for "offline dictation Mac." That doesn't affect the product, but it's why you might have found them.

## Bottom Line

<Callout type="tip">
**Developers:** Whisperer. JesType doesn't have Code Mode or per-app profiles.

**General dictation:** Both work. Whisperer has more features (AI modes, file transcription, history). JesType is simpler if that's what you want.

**Privacy:** Tie. Both run offline.

**Value:** Same price, more features in Whisperer. Code Mode, AI post-processing, per-app profiles, personal dictionary, file transcription, and history are all in the $14.99 Pro Pack.
</Callout>
