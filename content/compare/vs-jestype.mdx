JesType is an offline dictation app for Mac known for its simple UX and Moonshine model support. Priced at €14.95 one-time, it's in a similar price range to Whisperer. But the feature sets are quite different.

## Price Comparison

| | Whisperer | JesType |

| Price | $2.99 base / $14.99 Pro Pack | €14.95 one-time |
| Subscription | No | No |
| Cloud fees | None | None |
| Model | One-time purchase | One-time purchase |

<Callout type="info">
Both apps use a one-time purchase model at similar price points (~$15). The price difference comes down to features — Whisperer's Pro Pack includes Code Mode, per-app profiles, and AI post-processing that JesType doesn't offer.
</Callout>

## Where Whisperer Wins

<FeatureGrid>
<FeatureCard title="Code Mode" icon="code">
JesType is a general-purpose dictation app. Whisperer's Code Mode adds voice commands for `camelCase`, `snake_case`, `PascalCase`, programming symbols (parentheses, brackets, arrows, semicolons), and intelligent code spacing. Essential for developers.
</FeatureCard>

<FeatureCard title="Three Engines" icon="cpu">
Whisperer supports Whisper (Metal GPU), Parakeet (Neural Engine), and Apple Speech. JesType supports Whisper and Moonshine. Whisperer's Parakeet engine is the fastest option on Apple Silicon — it runs on the Neural Engine, leaving the GPU free.
</FeatureCard>

<FeatureCard title="AI Post-Processing" icon="sparkles">
Whisperer includes 10 built-in AI modes (rewrite, translate, format, summarize, grammar, and more) — all running offline on your Mac. JesType transcribes but doesn't transform text.
</FeatureCard>

<FeatureCard title="Per-App Profiles" icon="target">
Automatic dictation mode switching per app. Code Mode in VS Code, natural language in Slack, email formatting in Gmail. JesType uses the same settings for every app.
</FeatureCard>
</FeatureGrid>

## Where JesType Wins

### Moonshine Model Support
JesType supports the Moonshine transcription model, which Whisperer does not. Moonshine is optimized for fast, lightweight transcription and may be preferred by users who want the smallest possible model footprint.

### Simple UX
JesType has a deliberately minimal interface focused on core dictation. If you want a simple, no-frills dictation tool without developer features, JesType's simplicity is an advantage.

### Strong SEO Landing Page
JesType has built effective SEO landing pages targeting "offline dictation Mac" — they rank well for this keyword. Credit where due.

## The Verdict

<Callout type="tip">
**For developers:** Whisperer is the clear choice. Code Mode, per-app profiles, and AI post-processing are features JesType simply doesn't have.

**For general dictation:** Both are solid offline options at ~$15. Whisperer offers more features (AI modes, file transcription, history, filler word removal). JesType is simpler.

**For privacy:** Both are 100% offline — a tie on privacy.

**For value:** At the same price point, Whisperer includes significantly more features. Code Mode, AI post-processing (10 modes), per-app profiles, personal dictionary, file transcription, and transcription history — all included in the $14.99 Pro Pack.
</Callout>
