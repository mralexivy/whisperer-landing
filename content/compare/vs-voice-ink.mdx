VoiceInk and Whisperer are both Mac dictation apps with one-time pricing and on-device processing. They share the same foundation — whisper.cpp models running locally — but differ in features and target audience.

## Price Comparison

| | Whisperer | VoiceInk |
|---|-----------|---------|
| Base app | Free | One-time purchase |
| Pro features | $14.99 one-time | Included |

Both apps avoid subscriptions, which is refreshing in a market moving toward monthly fees. Pricing is comparable.

## Where Whisperer Wins

<FeatureGrid>
<FeatureCard title="Code Mode" icon="code">
The single biggest differentiator. Whisperer lets you dictate `camelCase`, `snake_case`, `CONSTANT_CASE`, parentheses, brackets, semicolons, and other programming symbols by voice. VoiceInk has no code-specific features.
</FeatureCard>

<FeatureCard title="Per-App Profiles" icon="target">
Whisperer switches dictation behavior based on the active app. Code Mode for your IDE, natural language for email, chat style for Slack. VoiceInk uses the same mode everywhere.
</FeatureCard>

<FeatureCard title="Personal Dictionary" icon="book-open">
Add project-specific terms, colleague names, and technical jargon to improve recognition accuracy. VoiceInk relies solely on the base Whisper model vocabulary.
</FeatureCard>
</FeatureGrid>

## Where VoiceInk Wins

### Simplicity
VoiceInk focuses on doing one thing well — clean, simple dictation. If you don't need developer features or per-app profiles, VoiceInk's simpler interface might appeal to you.

### All Features Included
VoiceInk includes all features in the base purchase — no separate "Pro Pack" upgrade needed.

## The Verdict

<Callout type="tip">
**For developers:** Whisperer is the clear winner with Code Mode, per-app profiles, and the personal dictionary.

**For general users:** Both are solid choices. Whisperer offers more features with the Pro Pack, while VoiceInk is simpler and more straightforward.

**For budget-conscious users:** Both are one-time purchases at similar price points. Whisperer's free tier lets you try before buying.
</Callout>
