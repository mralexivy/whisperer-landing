Both apps run Whisper locally. Both cost a one-time fee. VoiceInk is simpler. Whisperer has developer features and costs less.

## Price

| | Whisperer | VoiceInk |
|---|-----------|---------|
| Base app | Free | One-time purchase |
| Pro features | $14.99 one-time | Included |

No subscriptions on either. Both one-time purchases.

| Feature | Whisperer | VoiceInk |

## Where Whisperer Wins

<FeatureGrid>
<FeatureCard title="Code Mode" icon="code">
Dictate `camelCase`, `snake_case`, parentheses, brackets, semicolons. VoiceInk can't do this.
</FeatureCard>

<FeatureCard title="Per-App Profiles" icon="target">
Code Mode in your IDE, normal text in Slack. Automatic switching. VoiceInk is the same everywhere.
</FeatureCard>

<FeatureCard title="Personal Dictionary" icon="book-open">
Add project names, colleague names, technical terms. VoiceInk uses Whisper's built-in vocabulary only.
</FeatureCard>
</FeatureGrid>

## Where VoiceInk Wins

### Simplicity
VoiceInk is simpler. No modes, no profiles. If you don't need developer features, that simplicity might be what you want.

### No Pro Upgrade
All features included in the base price. No second purchase.

## What Whisperer Adds

<FeatureGrid>
<FeatureCard title="AI Post-Processing" icon="sparkles">
[Ten AI modes](/features/ai-writing) that run offline: rewrite, translate, format, summarize, fix grammar. VoiceInk just transcribes.
</FeatureCard>
<FeatureCard title="Three Engines" icon="cpu">
[Whisper (GPU), NVIDIA (Neural Engine), Apple Speech](/features/offline-transcription). Switch without restarting. VoiceInk has Whisper only.
</FeatureCard>
<FeatureCard title="File Transcription" icon="folder-open">
Drag audio or video files to [transcribe them](/features/file-transcription). VoiceInk doesn't do file transcription.
</FeatureCard>
<FeatureCard title="History" icon="clock">
Every transcription saved. Search, playback, stats. VoiceInk doesn't keep history.
</FeatureCard>
</FeatureGrid>

## Bottom Line

<Callout type="tip">
**Developers:** Whisperer. [Code Mode](/code-mode), per-app profiles, personal dictionary.

**General users:** Both work. Whisperer has more features (AI, file transcription, history, three engines) and costs less ($14.99 vs ~$25).

**Budget:** Whisperer is cheaper. Free trial lets you test first.
</Callout>
