VoiceInk and Whisperer are both Mac dictation apps with one-time pricing and on-device processing. They share the same foundation — whisper.cpp models running locally — but differ in features and target audience.

## Price Comparison

| | Whisperer | VoiceInk |
|---|-----------|---------|
| Base app | Free | One-time purchase |
| Pro features | $14.99 one-time | Included |

Both apps avoid subscriptions, which is refreshing in a market moving toward monthly fees. Pricing is comparable.

## Where Whisperer Wins

<FeatureGrid>
<FeatureCard title="Code Mode" icon="code">
The single biggest differentiator. Whisperer lets you dictate `camelCase`, `snake_case`, `CONSTANT_CASE`, parentheses, brackets, semicolons, and other programming symbols by voice. VoiceInk has no code-specific features.
</FeatureCard>

<FeatureCard title="Per-App Profiles" icon="target">
Whisperer switches dictation behavior based on the active app. Code Mode for your IDE, natural language for email, chat style for Slack. VoiceInk uses the same mode everywhere.
</FeatureCard>

<FeatureCard title="Personal Dictionary" icon="book-open">
Add project-specific terms, colleague names, and technical jargon to improve recognition accuracy. VoiceInk relies solely on the base Whisper model vocabulary.
</FeatureCard>
</FeatureGrid>

## Where VoiceInk Wins

### Simplicity
VoiceInk focuses on doing one thing well — clean, simple dictation. If you don't need developer features or per-app profiles, VoiceInk's simpler interface might appeal to you.

### All Features Included
VoiceInk includes all features in the base purchase — no separate "Pro Pack" upgrade needed.

## Feature Deep Dive: What Whisperer Adds

Beyond Code Mode and per-app profiles, Whisperer includes several features VoiceInk doesn't:

<FeatureGrid>
<FeatureCard title="AI Post-Processing" icon="sparkles">
[10 offline AI modes](/features/ai-writing/): Rewrite, Translate, Format, Summarize, Grammar, List Format, Coding, Email, Creative, and Custom. All on-device. VoiceInk transcribes but doesn't transform text.
</FeatureCard>
<FeatureCard title="Three Engines" icon="cpu">
[Whisper (Metal GPU), Parakeet (Neural Engine), Apple Speech](/features/offline-transcription/). Hot-swap without restarting. VoiceInk uses Whisper only. Parakeet's CTC vocabulary boosting is unique to Whisperer.
</FeatureCard>
<FeatureCard title="File Transcription" icon="folder-open">
Drag an audio or video file into Whisperer to [transcribe it offline](/features/file-transcription/). VoiceInk doesn't support file transcription.
</FeatureCard>
<FeatureCard title="Transcription History" icon="clock">
Every transcription is saved with [full-text search, audio playback, and usage statistics](/features/transcription-history/). VoiceInk doesn't maintain history.
</FeatureCard>
</FeatureGrid>

## The Verdict

<Callout type="tip">
**For developers:** Whisperer is the clear winner with [Code Mode](/code-mode/), [per-app profiles](/blog/dictation-for-per-app-profiles/), and the [personal dictionary](/features/personal-dictionary/).

**For general users:** Both are solid choices. Whisperer offers substantially more features — AI post-processing, file transcription, transcription history, three engines — at a lower price ($14.99 vs ~$25).

**For budget-conscious users:** Whisperer is actually cheaper ($14.99 vs ~$25) and has more features. The free trial lets you try before buying.
</Callout>
